# Vintage Photos

Drop black-and-white football photos here (any size, any format —
jpg, png, webp, gif). The paper picks a few at random per issue and
renders them as uniform grayscale newspaper plates.

Aim for ~20 images for good variety. If this folder is empty, the
paper shows a halftone-dot placeholder instead, so nothing breaks.

Note: this is a privately-shared file for your league. You are
responsible for what you put here.
